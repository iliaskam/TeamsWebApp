TeamsWebApp

This ASP.NET web application displays users in a grid and their chats in a popup window. 

- The application shows dummy data that is imported to the database using SQL Server. 
- The database was created using Visual Studio's migration.
- The Users page of the application presents a list of users retrieved from the database.
- To create the popup window, i used bootstrap modal dialog.
- Click on the "Show Chats" button in the Users page to open the popup window.
- Select a date range to display all chats between users within the specified date range.
